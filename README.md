# 好想安静地写代码

一个个人小游戏作品集网站。

## 🎮 游戏列表

### 方块消消乐
九种颜色方块下落，相同颜色连在一起即可消除得分。支持自定义画布大小、难度、方块图片、背景图片等。

## 📁 项目结构

```
├── index.html          # 主页
├── home-style.css      # 主页样式
├── home-script.js      # 主页交互（玩法说明弹窗）
└── games/
    └── fangkuai/       # 方块消消乐游戏
        ├── index.html
        ├── style.css
        └── script.js
```

## 🚀 操作方式（方块消消乐）

- `A` - 向左移动方块
- `D` - 向右移动方块
- `S` - 加速下落
- `空格` - 切换方块颜色
